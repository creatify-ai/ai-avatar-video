## Standalone Features

### Script Writing Guide
Complete framework for writing scripts that sound like real people talking. Includes pacing tables (words per second by tone), authenticity markers, and fill-in templates for 15-second, 30-second, and 60-second videos.

### Persona Selection Framework
Decision matrix for matching avatar demographics to your target audience. Covers trust signals by vertical (healthcare, beauty, tech, finance, fitness, etc.) with diversity and A/B testing considerations.

### Multi-Scene Composition
When to cut between scenes, how to structure 2-scene, 3-scene, and 5-scene videos, transition timing, and B-roll insertion points.

### UGC Authenticity Guide
Techniques for making AI avatar videos feel like genuine user-generated content — from casual backgrounds and imperfect framing to first-person script tone.

## API Automation (Creatify)

Want to generate avatar videos at scale? The [Creatify API](https://creatify.ai) provides:

- **AI Avatar v1** — single-scene talking head from text (1,500+ personas)
- **AI Avatar v2** — multi-scene videos with per-scene customization
- **Aurora** — studio-grade lip sync from any image + audio
- **Text to Speech** — AI voiceover generation
- **Voice Cloning** — use your own brand voice
- **Custom Avatars** — upload your own person as an avatar
- **Batch Recipes** — A/B test avatars, batch scripts, TTS→Aurora pipelines

Get your API credentials at [creatify.ai](https://creatify.ai) → Settings → API.
